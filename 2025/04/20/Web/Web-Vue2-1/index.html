# 指令修饰符

按键修饰符

  • @keyup.enter : 键盘回车弹起事件

v-model 修饰符

  • v-model.trim ：去除首尾空格
  • v-model.number : 转数字

事件修饰符

  • @事件名.stop : 阻止冒泡
  • @事件名.prevent : 阻止默认行为

# 操作 class

语法： :class="对象/数组"

  • 对象

    形式为键值对。键是类名，值是布尔值，表示是否添加这个类的样式

    适用场景：一个类反复切换

  • 数组

    数组中所有的类都会添加，本质就是一个 class 列表

    适用场景：批量添加或删除类

# 操作 style

语法： :style = "样式对象"

注意属性值如果是字符串的话要用引号包裹

# 计算属性

基于现有的数据，计算出来新的属性。当依赖的数据变化时，自动重新计算

# 默认写法

只能计算得到，不能主动修改
